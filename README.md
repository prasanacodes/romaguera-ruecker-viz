# Customer Journey Alluvial Diagram

**Author**: [22f1001255@ds.study.iitm.ac.in](mailto:22f1001255@ds.study.iitm.ac.in)

## Project Overview
This repository contains a strategic visualization for **Romaguera Ruecker**, demonstrating the customer journey flow from initial channel awareness to final purchase. The visualization highlights conversion rates and drop-off points across the marketing funnel.

## Visualization
The included `chart.png` is an Alluvial Diagram generated to map the volume of customer flow between the following stages:
1.  **Acquisition Channels** (Search, Social, Email, Direct, Referral)
2.  **Awareness**
3.  **Interest**
4.  **Consideration**
5.  **Purchase** (or Loss)

## Tools Used
- **RAWGraphs**: For generating the Alluvial flow visualization.
- **Data Source**: Modeled customer touchpoint data.
